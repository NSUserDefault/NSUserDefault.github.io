4、安装主题（Next为例）

更多主题查看官网https://hexo.io/themes/
知乎推荐主题https://www.zhihu.com/question/24422335

复制主题

Hexo 安装主题的方式非常简单，只需要将主题文件拷贝至站点目录的 themes 目录下， 然后修改下配置文件即可

启用主题
打开根目录下的_config.yml文件，找到 theme 字段，并将其值更改为 next

更改主题外观

NexT有三个外观，直接更改主题_config.yml文件的 scheme 参数即可，如果显示的是繁体中文，那么站点配置文件中的 language: zh-CN

再执行 hexo clean 和 hexo s 就可以在本地预览效果了
